# COMP3610A2

## Assignment 2: ML Model Training & Evaluation

This notebook demonstrates a complete machine learning workflow using NYC Yellow Taxi data. The main steps include:

1. **Data Cleaning:** Download, validate, and clean raw taxi trip data.
2. **Feature Engineering:** Create new features (duration, speed, boroughs, etc.) and preprocess data.
3. **Model Training:** Train and evaluate regression and classification models (Linear/Logistic Regression, Random Forest, Neural Network) to predict tip amount and high-tip trips.
4. **Model Evaluation:** Compare model performance using metrics and plots, and analyze feature importance.

**Technologies used:** Python, Polars, Pandas, Scikit-learn, PyTorch, Matplotlib, Seaborn.

**How to run:**
- Install dependencies from `requirements.txt`.
- Open and run the cells in `assignment2.ipynb` sequentially.

This project is for COMP3610 Assignment 2.
